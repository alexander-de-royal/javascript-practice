1. Adjacent selector or Combinator selector

Select only the paragraphs that are immediately preceded by an h1

h1 + p {color: white}

2. Direct Child

Select only the li that are direct children of div element

div > li {color: white}

3. Attribute Selector

Select all input elements where the type attribute is set to "text"

input[type="text"]{width: 300px; color: yellow}

4. Pseudo Classes

keyword added to a selector that specifies a special state of the selected elements

:active :checked :first :first-child :hover :not() :nth-child() :nth-of-type()

5. Pseudo Elements

Keyword added to a selector that lets you style a particular part of selected element(s)

::after ::before ::first-letter ::first-line ::selection

6. Specificity

is how the browser decides which rules to apply when multiple rules could apply to the same element
